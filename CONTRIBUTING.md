# Contributing

Thanks for improving this template.

## Scope

We welcome:

- Portability fixes (Linux/macOS shell runners, `PATH`, `claude` detection).
- Clearer English docs and troubleshooting.
- Security hygiene (no real tokens in examples).

## Process

1. Open an issue or MR describing the change.
2. Do not commit real API keys, trigger tokens, or Feishu secrets.
3. Keep changes focused on CI, webhook, and documentation.

## Out of scope

- Proprietary business logic or internal-only paths unless behind documented variables.
